
# Bananagrams

This project was bootstrapped with [Vite](https://vitejs.dev/) using the React + TypeScript template.

## Features

- React + TypeScript
- Vite for fast development
- Tailwind CSS (with @tailwindcss/vite plugin)
- Prettier with prettier-plugin-tailwindcss
- ESLint (warns on unused variables)

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Formatting

Prettier is configured with the Tailwind CSS plugin. Run Prettier to auto-format your code:

```bash
npx prettier --write .
```

## Linting

ESLint is set up to warn (not error) on unused variables in TypeScript files.

## Customization

- Tailwind CSS is imported in `src/index.css`.
- Project title is set to "Bananagrams" in `index.html`.

---
No Electron setup is included.
